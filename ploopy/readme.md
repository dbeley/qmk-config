# My own Ploopy keymap

Only tested on a left-handed ploopy classic trackball.

3 layouts:

Layout 0
```
              .-------.    .---------.
 ._________. /         \  /      /    \
/LAY2/LAY1/ |  BALL    | /      /      \
\BTN5\BTN3\ \         / /  .------.    |
 \    \    \ '-------' /  | BTN3  |   /
  \    \    \         /   '------'   /
   '--------'        / BTN1 / BTN2  /
                    '------'--------'
```

Layout 1
```
              .-------.    .---------.
 ._________. /         \  /      /    \
/DPI /####/ |  BALL    | /      /      \
\    \    \ \         / /  .------.    |
 \    \    \ '-------' /  | DRSCR |   /
  \    \    \         /   '------'   /
   '--------'        / PGUP / PGDWN /
                    '------'--------'
```

Layout 2
```
              .-------.    .---------.
 ._________. /         \  /      /    \
/####/DRSC/ |  BALL    | /      /      \
\    \    \ \         / /  .------.    |
 \    \    \ '-------' /  | DRSCR |   /
  \    \    \         /   '------'   /
   '--------'        / NXTAB/ PRTAB /
                    '------'--------'
```

- DPI: cycles DPI
- DRSC/DRSCR: Drag Scroll
- NXTAB: Ctrl+Tab
- PRTAB: Ctrl+Shift+Tab
