# My own Ploopy keymap

Only tested on a left-handed ploopy classic trackball.

3 layouts:

Layout 0
```
              .-------.     .---------.
 .---------. /         \   /      /    \
/LAY2/LAY1/ (   BALL    ) /      /      \
\    \    \  \         / /  .------.    |
 \    \    \  '-------' /  | BTN3  |   /
  \    \    \          /   '------'   /
   '--------'         / BTN1 / BTN2  /
                     '------'-------'
```

Layout 1
```
              .-------.     .---------.
 .---------. /         \   /      /    \
/DPI /####/ (   BALL    ) /      /      \
\    \    \  \         / /  .------.    |
 \    \    \  '-------' /  | DRSCR |   /
  \    \    \          /   '------'   /
   '--------'         / PGUP / PGDWN /
                     '------'-------'
```

Layout 2
```
              .-------.     .---------.
 .---------. /         \   /      /    \
/####/DRSC/ (   BALL    ) /      /      \
\    \    \  \         / /  .------.    |
 \    \    \  '-------' /  | DRSCR |   /
  \    \    \          /   '------'   /
   '--------'         / BTN3 / DRSCR /
                     '------'-------'
```

- DPI: cycles DPI
- DRSC/DRSCR: Drag Scroll
