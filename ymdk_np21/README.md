# The default keymap for the YMDK NP21 keypad

The four top keys control the active layer. From left to right:
- qwerty gaming
- gaming (qwerty gaming with arrow keys)
- numpad
- qwerty

## Qwerty gaming

- L_QG: Qwerty gaming layer (this layer)
- L_Ga: Gaming layer
- L_Nu: Numpad layer
- L_Qw: Qwerty layer
- Md: Alt qwerty gaming layer
